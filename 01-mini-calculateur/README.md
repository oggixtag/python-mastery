# 01 - Mini-Calculateur Python 🧮

Bienvenue dans le premier projet de la série **Python Mastery**. Ce projet consiste en une calculatrice interactive en ligne de commande (CLI) permettant d'effectuer des opérations arithmétiques de base avec une gestion d'erreurs robuste.

---

## 📝 Description

Ce script Python permet à l'utilisateur de réaliser des calculs simples de manière dynamique. Il a été conçu pour mettre en pratique les bases fondamentales du langage : les variables, les conditions, les boucles, et la gestion des exceptions.

## ✨ Fonctionnalités

* **Opérations supportées** : Addition, Soustraction, Multiplication, et Division.
* **Validation des saisies** : Le programme vérifie que l'utilisateur entre bien des nombres.
* **Sécurisation des calculs** : Gestion explicite de la division par zéro pour éviter tout plantage.
* **Interface interactive** : Possibilité d'enchaîner plusieurs calculs ou de quitter le programme proprement.

## 🚀 Installation et Utilisation

1.  **Cloner le dépôt** :
    ```bash
    git clone [https://github.com/oggixtag/python-mastery.git](https://github.com/oggixtag/python-mastery.git)
    ```
2.  **Accéder au dossier** :
    ```bash
    cd 01-mini-calculateur
    ```
3.  **Lancer l'application** :
    ```bash
    python main.py
    ```

## 🛠️ Technologies utilisées

* **Langage** : Python 3.x
* **Concepts** : `try...except`, `while loops`, `if/elif/else`.
